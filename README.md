# Tech IMS

A web-based **Computer Laboratory Equipment Inventory Management System** built with **Python**, **Django**, and **PostgreSQL**. This project was developed as a personal learning project to strengthen my backend development skills while building a practical inventory management application.

## Features

* User authentication and login system
* Role-based access control

  * Administrator
  * Staff
  * Viewer
* Equipment inventory management
* Add, edit, and delete equipment
* Borrow and return equipment
* Automatic equipment status updates
* Dashboard with inventory statistics
* Django Admin integration
* PostgreSQL database
